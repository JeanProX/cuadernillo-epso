CUADERNILLO DIGITAL EPSO — VERSIÓN SIN QR

1. Abre esta carpeta en Visual Studio Code.
2. Abre index.html.
3. Presiona Go Live.
4. Prueba en la computadora con http://127.0.0.1:5500
5. Para probar en un celular conectado a la misma red, usa la IP local de la computadora, por ejemplo:
   http://192.168.100.32:5500

VISTA PARA CÍRCULO DE LECTURA
- Al terminar, presiona “Ver preguntas y respuestas”.
- Se muestran únicamente las preguntas y las respuestas, sin las imágenes del cuadernillo.
- El botón “Imprimir / guardar como PDF” abre el diálogo del navegador; no descarga nada automáticamente.

PUBLICACIÓN EN GITHUB PAGES
- Sube index.html y las carpetas css, js e img a la raíz del repositorio.
- Activa GitHub Pages desde Settings > Pages.
- Comparte directamente el enlace público de GitHub Pages.

IMPORTANTE
- Las respuestas se guardan en localStorage, es decir, en el navegador y dispositivo de cada participante.
- No se envían a un servidor ni se comparten automáticamente.
